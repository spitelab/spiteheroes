Game Mechanics

- level script is loaded and interpreted
- unknown commands are ignored
- paths are drawn
- entities are placed
- level title and description is show to the user
- turn by turn cycle starts (no interactions for now)
- hero agent ai
- ghost agent ai
- obtainable object (key)
- openable item (chest, closed/open state)
- Win and level ends, next level is loaded
- Loose and level restarts
